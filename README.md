# Study Plan Analyzer

> **プロのプロジェクト管理術を、あなたの学習に最適化。**

> **進捗を可視化し、計画倒れを防ぐ 「迷わない完走へのロードマップ」**

---

##  アプリ概要

多くの学習管理アプリは「何時間勉強したか」という「時間（コスト）」の記録に留まります。しかし、本当に大切なのは「その時間でどれだけの成果を得られたか」です。

本アプリは、単なる学習記録ツールを超え、プロジェクト管理の手法である **EVM（Earned Value Management：アーンド・バリュー・マネジメント）** を個人学習に最適化して取り入れ、ユーザーのモチベーションと効率を最大化することを目指す進捗管理ツールです。

「予定より遅れているのか？」「今の集中力は高いのか？」を数値で客観的に可視化し、SPI（進捗効率）やCPI（時間効率）に基づいた的確な分析と自動リスケジュールを提供します。

---

##  主な機能

### 1. EVMを用いた多角的な進捗分析
プロジェクト管理のプロが使う指標を、学習者向けに直感的な言葉で提供します。
* **進捗スピード (SPI)**: 「立てた予定に対して、今どれだけ進んでいるか」を数値化。
* **集中効率 (CPI)**: 「1時間あたりの集中度（学習密度）」を算出。
* **完了予測日の自動導出**: 現在のペースから、現実的なゴール日をリアルタイムに自動計算。

### 2. 「納得感」を重視したダッシュボード
各指標カードをクリックすると「ドリルダウン表示」で、詳細なデータを確認できます。
* **計算根拠の開示**: 「なぜこの数字なのか」という根拠を、実績データとセットで解説。
* **パーソナライズ診断**: 現在の状態に基づいた具体的な改善アドバイスを提示。

### 3. 入力負荷を極限まで減らす「学習モード」
時間を細かく設定する手間を省き、すぐに学習を開始できる仕組みを備えています。
* **3つの学習ペース**: 「短期集中」「標準」「ゆったり」から選ぶだけで、週間目標時間を自動セット。
* **トピック・タスクの自動連携**: トピックを作成すると、計測に必要な最小単位のタスクを裏側で自動生成。

### 4. 計画倒れを防ぐ「自動リスケジュール」
計画が遅れた際の心理的負担を、システムによる自動調整で解消します。
* **ワンクリック再分配**: 遅れている未完了タスクを、明日以降の稼働予定に自動で割り振り。

### 5. 安心のデータ管理
* **LocalStorage保存**: 面倒なログイン不要。ブラウザを閉じてもデータは保持されます。
* **JSONエクスポート/インポート**: データのバックアップや、別デバイスへの引き継ぎに対応。

##  このアプリを使うメリット

### 1. 迷いのない「今日」の目標設定
プロジェクトを入力する過程で、自然とタスクの細分化が行われます。「次に何をすべきか」が明確な状態で1日をスタートできるため、実行に移るまでの迷いがなくなります。

### 2. 努力の「手応え」をリアルタイムに実感
タスクを完了するたびに進捗度が自動計算されます。「どれくらい進んだか」が常に最新の数値で示されるため、自分の頑張りが着実にゴールへ近づいている実感を得られます。

### 3. 客観的なデータによる「安心感」
日ごとの達成率や全体の進捗状況をグラフで可視化します。調子が良い時も悪い時も、感覚ではなくデータで状況を俯瞰できるため、根拠のない焦りや不安が解消されます。

### 4. 専属コーチのような「次の一手」の提案
独自の診断機能が、現在の遅れや効率を分析します。「学習時間を増やすべきか」「タスクをさらに分けるべきか」といった改善アクションをアプリが提示するため、分析に時間を使わず、すぐに軌道修正が可能です。

### 5. 計画倒れをゼロにする「リスケジュール」機能
計画が遅れても、ボタン一つで現状に即した計画へ引き直すことができます。
* **ポジティブな再出発**: リスケ後は「調整完了」としてSPIが1.0にリセットされます。過去の遅れに縛られず、「今日からまた頑張れる」仕組みを整えています。
